# Helpers

As clases de axuda que teñen breakpoints indica que se pode asignar un valor diferente segundo o tamaño de pantalla no
que esteamos a visualizar o elemento, por exemplo aplicando só unha marxe a partir de dispositivos *large* con
`has-margin-lg-2`.Para máis información consulta a
[documentación dos breakpoints de Bootstrap](https://getbootstrap.com/docs/5.3/layout/breakpoints/) e o noso Flexbox Grid.

## Variables Sass

```scss
$aspect-ratios: (
  "1x1": 100%,
  "4x3": calc(3 / 4 * 100%),
  "16x9": calc(9 / 16 * 100%),
  "21x9": calc(9 / 21 * 100%)
);

$spacer: 1rem;
$spacers: (
  0: 0,
  1: $spacer * .25,
  2: $spacer * .5,
  3: $spacer,
  4: $spacer * 1.5,
  5: $spacer * 3,
);
```
