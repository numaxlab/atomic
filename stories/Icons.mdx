# Iconas

Este framework inclúe a versión gratuíta da biblioteca de iconas de Font Awesome. Podes consultar a lista completa na
súa [páxina oficial](https://fontawesome.com/search?o=r&m=free).

Consulta tamén a [documentación](https://docs.fontawesome.com/web/setup/get-started) con detalles de uso e implementación.
