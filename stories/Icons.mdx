## Uso en Sass

Para usar iconas no teu Sass, emprega o placeholder `%fa-icon` (ou o mixin `fa-icon`) para configurar o elemento como icona. Logo emprega o
placeholder correspondente ao estilo de icona que precisas (`.fas`, `.far` ou `.fab`). Para rematar usa a función
`fa-content` e o nome da variable coa icona que precises.

```scss
  .at-user-icon {
    @extend %fa-icon;
    @extend .fas;

    &::before {
      content: fa-content($fa-var-user);
    }
  }

  .at-twitter-icon {
    @include fa-icon;
    @extend .fab;

    &::before {
      content: fa-content($fa-var-twitter);
    }
  }
```

## Variables Sass

```scss
$fa-css-prefix          : fa;
$fa-style               : 900;
$fa-style-family        : "Font Awesome 6 Free";

$fa-display             : inline-block;

$fa-fw-width            : fa-divide(20em, 16);
$fa-inverse             : #fff;

$fa-border-color        : #eee;
$fa-border-padding      : .2em .25em .15em;
$fa-border-radius       : .1em;
$fa-border-style        : solid;
$fa-border-width        : .08em;

$fa-size-scale-2xs      : 10;
$fa-size-scale-xs       : 12;
$fa-size-scale-sm       : 14;
$fa-size-scale-base     : 16;
$fa-size-scale-lg       : 20;
$fa-size-scale-xl       : 24;
$fa-size-scale-2xl      : 32;

$fa-sizes: (
  "2xs"                 : $fa-size-scale-2xs,
  "xs"                  : $fa-size-scale-xs,
  "sm"                  : $fa-size-scale-sm,
  "lg"                  : $fa-size-scale-lg,
  "xl"                  : $fa-size-scale-xl,
  "2xl"                 : $fa-size-scale-2xl
);

$fa-li-width            : 2em;
$fa-li-margin           : $fa-li-width * fa-divide(5, 4);

$fa-pull-margin         : .3em;

$fa-primary-opacity     : 1;
$fa-secondary-opacity   : .4;

$fa-stack-vertical-align: middle;
$fa-stack-width         : ($fa-fw-width * 2);
$fa-stack-z-index       : auto;

$fa-font-display        : block;
$fa-font-path           : "../webfonts";
```

Consulta o uso de cada variable na [documentación oficial](https://docs.fontawesome.com/web/use-with/scss/#simplify-and-customize-with-variables).
