# Installation

Primeiro instala Atomic Framework na túa app con npm:

```
npm install @numaxlab/atomic
```

Unha vez o tes instalado podes escoller dúas formas para empregalo, extendelo ou adaptalo.

## Versión compilada e lista para usar

Hai unha versión compilada do framework preparada no paquete npm. Podes atopala en
```@numaxlab/atomic/dist/css/style.css``` xunto con ```@numaxlab/atomic/dist/css/fonts.css``` que contén as
tipografías necesarias. Podes copiar estes arquivos ao teu proxecto e incluílos na cabeceira
do teu HTML:
```html
<link href="/css/numaxlab-atomic/fonts.css" rel="stylesheet">
<link href="/css/numaxlab-atomic/icons.css" rel="stylesheet">
<link href="/css/numaxlab-atomic/style.css" rel="stylesheet">
```

## Sass

Se queres un maior nivel de adaptación podes usar a fonte Sass a través de dúas opcións. Fai o seguinte no arquivo
```custom.scss``` (nome de exemplo):

1. Importa o framework completo

```scss

// Include any default variable overrides here (though functions won't be available)

@import "@numaxlab/atomic/src/style";

// Then add any customization or additional code here

```

2. Importa só as partes necesarias

```scss

// 1. Import functions
@import "@numaxlab/atomic/src/utilities/functions";

// 2. Include any default variable overrides here

// 3. Initialize framework
@import "@numaxlab/atomic/src/utilities/init";

// 4. Import base styles
@import "@numaxlab/atomic/src/base/all";

// 5. Import the needed components only

// 6. Import your custom components or your project component extending the framework ones

```
