# Installation

[1] Instala Atomic Framework na túa app con npm.

```
npm install @numaxlab/atomic
```

[2] Configura Vite.

Engade o plugin de Vite @tailwindcss/vite á configuración.

```js
import { defineConfig } from 'vite';
import tailwindcss from '@tailwindcss/vite';

export default defineConfig({
  plugins: [
    tailwindcss(),
  ],
});
```

[3] Immporta Atomic Framework nos teus arquivos CSS e Javascript.

Engade os @import ao teu arquivo CSS para que importe Atomic Framework. Adicionalmente, importa as tipografías e iconas de
Atomic e indícalle a Tailwind CSS que directorios ten que escanear no uso de clases de utilidades.

```css
@import '@numaxlab/atomic/src/css/atomic.css';
@import '@numaxlab/atomic/src/css/fonts.css'; // Opcional, se queres usar as fontes de Atomic
@import '@numaxlab/atomic/src/css/icons.css'; // Opcional, se queres usar as iconas de Atomic

@source "../**/*.blade.php"; // Para escanear as plantillas Blade nun proxecto Laravel
```
