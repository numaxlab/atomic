# NUMAX Lab Atomic Framework

[Bootstrap](https://getbootstrap.com/) based, our CSS framework takes a minimal part of Bootstrap and customizes it to
our needs following the [Atomic Design](https://atomicdesign.bradfrost.com/) principles.

## Requirements

There are some important global styles and settings that are required, check
[Boostrap documentation](https://getbootstrap.com/docs/5.3/getting-started/introduction/#important-globals) for more
information.
