## Variables Sass

```scss
$white: #FFFFFF;
$gray-400: #E5E5E5;
$gray-600: #9D9D9D;
$gray: $gray-400;
$black: #000000;
$green: #487629;
$yellow: #FFC107;
$yellow-100: tint-color($yellow, 80%);
$red: #FF0000;

$grays: (
        "400": $gray-400,
        "600": $gray-600,
);

$colors: (
        "red": $red,
        "yellow": $yellow,
        "green": $green,
        "black": $black,
        "white": $white,
        "gray": $gray,
);

// See https://www.w3.org/TR/WCAG20/#visual-audio-contrast-contrast
$min-contrast-ratio: 4.5;

$primary: #0000FF;
$secondary: $gray-600;
$success: $green;
$warning: $yellow;
$danger: $red;

$theme-colors: (
        "primary": $primary,
        "secondary": $secondary,
        "success": $success,
        "warning": $warning,
        "danger": $danger,
);

$theme-colors-text: $theme-colors;

$theme-colors-bg-subtle: (
        "primary": tint-color($primary, 90%),
        "secondary": $white,
        "success": tint-color($success, 90%),
        "warning": tint-color($warning, 90%),
        "danger": tint-color($danger, 90%),
);

$theme-colors-border-subtle: $theme-colors;
```
